Sobre os meios de Leitura Digital:
- 56% dos leitores utilizam Celular ou Smartphone
- 49% dos leitores utilizam Computador
- 18% dos leitores utilizam Tablet
- 4% dos leitores utilizam E-reader

Os locais mais utilizados para leitura são:

- Casa
- Sala de aula
- Trabalho

Os locais mais indicados para cada dispositivo:

- E-reader: Ambientes externos e internos durante o dia.
Caso o e-reader ofereça iluminação própria, também é indicado para uso noturno.

- Tablet, Celular ou Computador: Ambientes internos como casa ou escritório.
